# Sweet Crumbs Bakery Website

Frontend website project for a bakery brand with public pages, dashboard pages, Firebase setup, and organized visual assets.

## Structure

- `index.html` and `login.html`: main entry pages
- `css/`: global styles, variables, animations, responsive rules, dashboard styles
- `js/`: app logic, animation, auth, Firebase, dashboard, products, utilities
- `assets/`: images, SVGs, fonts, and videos
- `pages/`: dashboard, products, orders, messages, gallery, settings
- `firebase/`: Firebase configuration and Firestore rules
